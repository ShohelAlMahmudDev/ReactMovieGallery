# ReactMovieGallery

## Overview

**ReactMovieGallery** is a dynamic and responsive web application built using **React** for the frontend and **Python** for backend services. The app leverages the **Movie Gallery API** to fetch and display information about movies in an intuitive and user-friendly interface.

## Features

- **Movie Search**: Search for movies by title or keyword.
- **Dynamic Gallery**: Displays movie posters, titles, and descriptions in a responsive grid layout.
- **API Integration**: Fetches real-time data from the Movie Gallery API.
- **Interactive UI**: Provides sorting, filtering, and other interactive features.

## Technologies Used

- **Frontend**: React, CSS, JavaScript
- **Backend**: Python (Flask/Django)
- **API**: Movie Gallery API

## Prerequisites

- Node.js (16.x or higher)
- Python (3.9 or higher)
- A valid Movie Gallery API key

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ShohelAlMahmudDev/ReactMovieGallery.git
   ```

2. Navigate to the project directory:
   ```bash
   cd ReactMovieGallery
   ```

3. Install frontend dependencies:
   ```bash
   npm install
   ```

4. Set up the backend (if applicable) and install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

5. Create a `.env` file in the root directory and add your Movie Gallery API key:
   ```env
   REACT_APP_API_KEY=your_api_key_here
   ```

6. Start the development server:
   ```bash
   npm start
   ```

## Usage

1. Open the application in your browser at `http://localhost:3000`.
2. Use the search bar to find movies by title or keyword.
3. Explore the gallery to view details like movie posters, descriptions, and ratings.
